
# Методология v2

Штрафы не являются киберугрозой. В модели они показываются как денежная стоимость правового несоответствия.

Правовой риск:

```math
R_{legal}=P_{gap}\times P_{detect}\times(F+C_{fix}+C_{law})
```

В интерфейсе добавлена таблица штрафных сценариев:

- неуведомление о намерении обрабатывать ПДн;
- неуведомление об утечке;
- утечка ПДн малого/среднего масштаба;
- утечка ПДн крупного масштаба;
- повторная утечка с оборотным штрафом.

В модели нет ROI. Используются только защитные затраты и снижение остаточного риска.
