
# Щит-МСП v2: штрафы видимы

Версия добавляет отдельный блок **«Возможные штрафы и правовая стоимость несоблюдения»**.

## Запуск

```bash
pip install -r requirements.txt
streamlit run app.py
```

или на Windows:

```bash
python -m streamlit run app.py
```

## Что добавлено

- таблица возможных штрафных сценариев по ПДн;
- диапазон штрафа;
- ожидаемый штрафной ущерб;
- правовой разрыв;
- коэффициент выявления;
- оборотный штраф как отдельный сценарий.

Расчёт не является юридическим заключением.
