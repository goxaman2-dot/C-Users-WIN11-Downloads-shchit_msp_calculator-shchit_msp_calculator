
# Риски ИБ - МСП 2026 v.3.0

Python/Streamlit-калькулятор остаточного информационного риска для МСП.

## Главное изменение v.3.0

Добавлен **4-й слой**:

> Социо-правовой слой штрафов, жалоб и публичности.

Он отделён от технических атак и организационных мер. В него входят:

- возможные штрафы;
- правовой разрыв;
- жалобы субъектов персональных данных;
- публичность утечки;
- регуляторный хвост;
- стоимость несоблюдения норм права.

## Запуск

```bash
pip install -r requirements.txt
python -m streamlit run app.py
```

## Модель

В калькуляторе четыре слоя:

1. технический;
2. организационно-правовой;
3. экономико-управленческий;
4. социо-правовой слой штрафов, жалоб и публичности.

В модели нет ROI. Используются только защитные затраты, остаточный риск и предотвращаемый ущерб.
